VCS - Version Control System
Это программа, которая позволяет создавать контрольные точки исходных кодов,
и в любой момент возвращаться к этим контрольным точкам.

Git/GitHub
Git != Github
Git - это система контроля версий
GitHub - это облачное хранилище

Repository - это хранилище исходных кодов.

fetch - скачивает изменения с удаленного репозитория на локальный;
pull  - скачивает и распаковывает изменения с удаленного репозитория на локальный;
push  - отправлет изменения с локального репозитория на удаленный;

COMMIT - это контрольная точка, в которую сохраняется снимок репозитория,
		 т.е., его текущее состояние на момент создания этой контрольной точки 
		 (выполнения COMMIT-а)
Commit message - содержит краткое описание измений в репозитории, 
				 либо какая цель была достигнута в процессе работы.

Branch - это последовательность Commit-ов
master - это главная ветка, которая неявно создается при создании репозитория.

merge - слияние веток
